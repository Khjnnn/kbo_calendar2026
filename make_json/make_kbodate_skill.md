# KBO 경기 일정 이미지 → JSON 변환 스킬

## 개요

사용자가 KBO 공식 홈페이지 또는 기타 출처의 **경기 일정표 이미지**를 제공하면,
이미지를 분석하여 `public/data/games_YYYY_MM.json` 형식의 JSON 파일을 생성한다.

## 입력

- KBO 경기 일정이 포함된 **이미지 파일** (스크린샷, 캡처, 사진 등)
- 이미지에는 보통 날짜, 요일, 홈팀, 원정팀, 경기 시간, 경기장 정보가 포함되어 있음

## 출력 형식

파일명: `games_YYYY_MM.json` (예: `games_2026_04.json`)
저장 경로: `public/data/`

```json
{
    "month": "YYYY-MM",
    "games": [
        {
            "date": "YYYY-MM-DD",
            "day": "요일(월/화/수/목/금/토/일)",
            "home": "홈팀명",
            "away": "원정팀명",
            "time": "HH:MM",
            "stadium": "경기장명"
        }
    ]
}
```

## 팀명 규칙

이미지에서 추출한 팀명은 아래 공식 약칭으로 **반드시 통일**한다:

| 공식 약칭 | 정식명칭 | 경기장 |
|-----------|---------|--------|
| LG | LG 트윈스 | 잠실 |
| KT | KT 위즈 | 수원 |
| SSG | SSG 랜더스 | 인천 |
| NC | NC 다이노스 | 창원 |
| 두산 | 두산 베어스 | 잠실 |
| KIA | KIA 타이거즈 | 광주 |
| 롯데 | 롯데 자이언츠 | 사직 |
| 삼성 | 삼성 라이온즈 | 대구 |
| 한화 | 한화 이글스 | 대전 |
| 키움 | 키움 히어로즈 | 고척 |

### 팀명 변환 예시
- "엘지" → "LG"
- "케이티" → "KT"
- "에스에스지" → "SSG"
- "엔씨" → "NC"
- "기아" → "KIA"

## 경기장명 규칙

홈팀 기준으로 경기장을 결정한다. 이미지에 경기장이 명시되어 있으면 그대로 사용하고,
없으면 홈팀의 기본 홈구장을 사용한다:

- LG, 두산 → "잠실"
- KT → "수원"
- SSG → "인천"
- NC → "창원"
- KIA → "광주"
- 롯데 → "사직"
- 삼성 → "대구"
- 한화 → "대전"
- 키움 → "고척"

## 변환 절차

1. **이미지 읽기**: 사용자가 제공한 이미지를 Read 도구로 확인한다.
2. **데이터 추출**: 이미지에서 아래 정보를 추출한다:
   - 해당 월 (YYYY-MM)
   - 각 경기의 날짜, 요일, 홈팀, 원정팀, 시간, 경기장
3. **팀명 정규화**: 추출한 팀명을 위 팀명 규칙표에 따라 공식 약칭으로 변환한다.
4. **경기장 보완**: 경기장 정보가 누락된 경우 홈팀 기준으로 자동 매핑한다.
5. **날짜 정렬**: 날짜순 → 같은 날짜 내에서는 시간순으로 정렬한다.
6. **JSON 생성**: 위 출력 형식에 맞춰 JSON 파일을 생성한다.
7. **저장**: `public/data/games_YYYY_MM.json` 경로에 저장한다.

## 주의사항

- **요일 표기**: 한글 한 글자 (월, 화, 수, 목, 금, 토, 일)
- **시간 형식**: 24시간제 `"HH:MM"` (예: `"14:00"`, `"18:30"`)
- **날짜 형식**: `"YYYY-MM-DD"` (예: `"2026-03-28"`)
- **더블헤더**: 같은 날 같은 팀이 2경기인 경우, 두 경기 모두 별도 항목으로 추가
- **올스타전/특별경기**: 일반 경기와 동일 형식으로 처리
- **우천취소**: 이미지에 취소 표시가 있으면 해당 경기는 제외
- **DH(지명타자)전**: 동일 카드가 같은 날 2경기면 시간만 다르게 기록
- 이미지 OCR이 불확실한 경우, 가장 가능성 높은 값을 사용하되 주석으로 불확실함을 알린다
- 기존 `public/data/` 폴더에 같은 월 파일이 있으면 **덮어쓰기 전 확인**을 요청한다

## 실행 예시

사용자: "4월 경기 일정 이미지야, JSON으로 만들어줘" + 이미지 첨부

→ 이미지를 분석하여 `public/data/games_2026_04.json` 생성
